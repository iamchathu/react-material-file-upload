# React Material File Upload

[React Dropzone][react-dropzone] based [Material UI][mui] file upload component for React.js. This is created for [Material UI][mui] v5 or later versions.
Written with Typescript. Exports `commonjs` and `esm` modules.

## Install

```bash
yarn add react-material-file-upload
```

- Peer dependacies.

Libary is depends on `@mui/material`, `@mui/icons-material`, `@emotion/react`, `@emotion/styled`. These are [Material UI][mui] dependacies.

[react-dropzone]: https://react-dropzone.js.org/
[mui]: https://mui.com
